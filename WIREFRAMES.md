# Wireframe Descriptions

## Hackathon Epic — Wireframe Description (For Codex)
- **Screen Name:** Hackathons
- **Purpose:** Help users discover hackathons and explore hackathon-built projects.
- **Layout Description:**
  - Header section with page title "Hackathons."
  - Top search bar centered at the top with placeholder text: "Search hackathons…"
  - Filter button group to the right of the search bar (filters: Date, Location, Format, College).
  - Left side: Large interactive map showing hackathon pins that covers roughly 60% of the width.
  - Right side: Vertical list titled "Upcoming Hackathons." Each item includes hackathon name, location, and date.
  - Below map and list: Horizontal section titled "Featured Hackathon Projects."
  - Under the title: A 4-card row showing project thumbnails with short titles.
  - A "View More" button centered below the project row.
- **Primary Components:** Search bar; filter buttons; interactive map; list of upcoming hackathons; row of featured projects.

## Profiles Epic — Wireframe Description (For Codex)
- **Screen Name:** User Profile
- **Purpose:** Display user identity, skills, links, and recent work.
- **Layout Description:**
  - Top-left: Circular profile photo.
  - Right of photo: User’s name (large font) with role or subtitle below it.
  - Full-width box under header for short bio text (2–3 sentences).
  - Skills section: Pill-shaped badges (JavaScript, React, C++, Figma, etc.).
  - Social links section with icon buttons for LinkedIn, GitHub, Figma, and a custom portfolio link.
  - Hackathon badges section showing badges for attended hackathons (DevFest, HackRPI, etc.) in a row.
  - Latest projects section: Two-column grid of project cards, each with project thumbnail, title, and a one-line description.
  - Bottom of the screen: "View Full Portfolio" button.
- **Primary Components:** Profile header; bio; skills tags; social icons; hackathon badges; latest projects grid.

## Portfolio Epic — Wireframe Description (For Codex)
- **Screen Name:** Portfolio
- **Purpose:** Display all user projects in grid format with case study detail view.
- **Layout Description:**
  - Header: "Portfolio" in bold at the top.
  - Filter bar under header with dropdowns for Category (AI, Web, Mobile, VR), Year, and Tech stack.
  - Main content: 3-column grid of project cards with thumbnail image, project title, tags (JavaScript, React, OpenAI, etc.), and a short (one-sentence) description.
  - Clicking a project opens a project detail view with large header image, title, "Built at: [Hackathon name]," problem description, solution description, and demo links (GitHub, Figma, Video).
- **Primary Components:** Filters; project grid; project detail view.
